2. 变量的命名
    - 变量名只能包含字母、数字、下划线
    - 变量不能包含空格
    - 关键字和函数名不能用做变量名
## 字符串
```
# name.py

name = "ada lovelace"
# 首字母大写
print(name.title())
# 全部转化为大写
print(name.upper())
# 全部转换为小写
print(name.lower)
# 合并字符串使用 +
last_name = "Hello"
print(last_name + name)

'''
string.rstrip()删除尾部空白
string.lstrip()删除头部空白
string.strip()删除两头空白
'''

```

## 元组
元组属于不可变序列
#### 元组的创建
1. 通过()创建元组
    ```python
   a=(1,2,3)
   或者 a = 1,2,3
   ```

2. 通过tuple()创建元组

   ```python
   b = tuple() #创建一个空元组对象
   b = tuple("abc")
   b = tuple(range(3))
   b = tuple([2,3,4])
   ```

   总结：

   ​	tuple()可以接受列表、字符串、其他序列类型、迭代器等生成元组

   ​	list()可以接受元组、字符串、其他序列类型、迭代器等生成列表
